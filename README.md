## Directory Structure
- `- android` Android Native Code
- `- ios` iOS Native Code
- `- src` App Code Base
  - `- components` Dumb Components
  - `- containers` Smart Components
  - `- redux` [ducks-modular-redux](https://github.com/erikras/ducks-modular-redux)
  - `- styles` App Styles
  - `- utils` APIs
  - `App.js`  Base Component
  - `Router.js` App Navigation
